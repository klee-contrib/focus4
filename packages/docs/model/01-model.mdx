# Modèle métier

Focus comprend un module complet pour gérer vos données métier. Il va vous permettre de :

-   Stocker vos données issues du serveur.
-   Charger vos données.
-   Afficher vos données
-   Editer vos données
-   Valider vos données.
-   Sauvegarder vos données.

## Définitions

### Entité

Une **entité** est un objet "métier", une répresentation d'une partie du modèle de l'application. Elle sera forcément un objet d'échange avec le serveur (un DTO) dans les diverses APIs qui vont exister entre les deux.

### Champ

Une entité peut constituée de **champs** et d'autres entités. Le champ est l'objet de base pour représenter **une** valeur.

_(Remarque : Un champ peut contenir n'importe quel type de valeur, tant que ce n'est pas une autre entité.)_

### Domaine

Le **domaine** d'un champ représente le **type de donnée métier** qui lui est associé (par exemple : une date, un numéro de téléphone, un montant...)
