# `ScrollspyContainer`

Le `ScrollspyContainer` est un composant du mise en page qui permet d'affiche un menu sur la gauche qui recapitule les titres de tous les [`Panel`](/docs/composants-focus4∕forms-panel--docs) posés à l'intérieur, avec la possibilité de naviger dessus en cliquant dessus.

Il **doit être placé dans un [`Scrollable`](/docs/mise-en-page-scrollable--docs)**, puisqu'il en a besoin pour placer son menu à gauche en sticky.
