# Mise en page

Les composants de mise en page Focus sont tous situés dans le module `@focus4/layout`. Vous n'êtes pas obligé de les utiliser dans votre application, mais un grand nombre de fonctionnalités, y compris dans d'autres modules, s'appuient sur ces composants.
