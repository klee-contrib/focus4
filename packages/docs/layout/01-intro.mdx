# Mise en page

Les composants de mise en page Focus sont tous situés dans le module `@focus4/layout`.

Si vous n'êtes pas obligés de tous les utiliser, un grand nombre de composants nécessitent au moins d'être posés dans le [`Layout`](/docs/mise-en-page-layout--docs) (ou au moins dans un [`Scrollable`](/docs/mise-en-page-scrollable--docs)) pour fonctionner correctement.
